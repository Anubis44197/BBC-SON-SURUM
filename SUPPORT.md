# Support

## Getting Help

For usage questions and troubleshooting:

1. Check `README.md` and `QUICK_START.md`.
2. Review `USER_GUIDE.md` for command-level guidance.
3. Use `python bbc.py audit .` for operational diagnostics.

## Issue Reporting

When opening an issue, include:

- BBC version/commit
- Operating system and Python version
- Exact command run
- Error output/log snippet
- Minimal reproduction steps

## Feature Requests

Feature requests are welcome. Please provide:

- Problem statement
- Proposed behavior
- Expected impact
- Backward compatibility considerations
